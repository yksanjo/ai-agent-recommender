## Deployment Links

### Railway (When Deployed)
After deploying to Railway, your app will be available at:
```
https://ai-agent-recommender-production.up.railway.app
```

To deploy:
1. Run `railway login`
2. Run `railway up`
3. Or connect via Railway dashboard: https://railway.app

### Streamlit Cloud (Alternative)
You can also deploy the UI to Streamlit Cloud:
- **Streamlit Cloud**: https://share.streamlit.io
- Connect your GitHub repo and deploy

## Local Development

### API Server
```bash
uvicorn src.api.main:app --reload
# Available at: http://localhost:8000
# API Docs: http://localhost:8000/docs
```

### Enhanced Web UI
```bash
streamlit run web/enhanced_app.py
# Available at: http://localhost:8501
```
